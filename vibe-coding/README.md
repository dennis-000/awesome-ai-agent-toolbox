# Vibe Coding Workflows & Resources 🚀✨

This directory will house best practices, configuration templates, and terminal scripts to speed up development with AI coding assistants (like Cursor, Claude Code, and Copilot).

## Planned Content

- **Agent Configurations:** Custom `.cursorrules` and prompt guidelines.
- **Shell Scripts & Hooks:** Automate local git commits, code linting, and formatting using AI.
- **Speed-Coding Tips:** Workflow diagrams showing how to build applications from scratch using natural language.
